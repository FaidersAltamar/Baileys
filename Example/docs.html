WhatsApp API Docs

Documentación interactiva de tu servidor REST Multi-Instancia.

POST /api/sessions/start Crear o reconectar instancia

Inicia una nueva instancia de WhatsApp en el servidor y la prepara para vincular un dispositivo.

Body (JSON)
ParámetroTipoDescripción
idstringNombre único de tu instancia (ej. "ventas", "soporte").
Ejemplo de petición
{
  "id": "ventas"
}
GET /api/sessions/:id/status Consultar estado y QR

Devuelve el estado de la conexión (connecting, connected, disconnected). Si está esperando vinculación, devolverá el código QR en Base64.

Parámetros de URL
ParámetroDescripción
idEl identificador único de la instancia.
POST /api/sessions/:id/send/text Enviar texto

Envía un mensaje de texto plano a un número a través de la instancia seleccionada.

Body (JSON)
ParámetroTipoDescripción
numberstringNúmero de destino con código de país (ej. "5215555555555").
messagestringEl texto del mensaje a enviar.
Ejemplo de petición
{
  "number": "5215555555555",
  "message": "¡Hola desde la API!"
}
POST /api/sessions/:id/send/media Enviar imagen o documento

Envía un archivo multimedia descargándolo de forma remota desde una URL pública.

Body (JSON)
ParámetroTipoDescripción
numberstringNúmero de destino con código de país.
urlstringURL pública de la imagen/documento (.png, .jpg, .pdf, etc).
captionstring(Opcional) Texto que acompaña la imagen/archivo.
Ejemplo de petición
{
  "number": "5215555555555",
  "url": "https://upload.wikimedia.org/wikipedia/commons/a/a7/React-icon.svg",
  "caption": "Logotipo de React"
}
DELETE /api/sessions/:id Cerrar sesión

Desvincula el dispositivo de WhatsApp asociado a la instancia y elimina permanentemente los archivos de sesión del servidor.

Parámetros de URL
ParámetroDescripción
idEl identificador único de la instancia a eliminar y desvincular.
